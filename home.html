About the Publisher Data

This data contains information about Amazon's ebooks. Some of this information is the genre of the book, the publisher, the book's rank, the book's revenue. The data was created in 2015 and records 54,000 ebooks, so so ebook data may be missing. This web page only focuses 2015 data. The data used in this web app comes from the CORGIS Dataset Project, it can be reached here.

Each book has its own set of data, but in that set there is not the book's title or the author. Instead it is organized by five categories, genre, publisher, sold by, statistics, and daily. Statistics and daily have sub-categories
